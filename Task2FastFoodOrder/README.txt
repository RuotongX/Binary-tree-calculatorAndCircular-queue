Just import this object into eclipse and run it, follow the hits while running.

If you want system run just type a name for new customer and the time will keep going, while you add new customer the old customer will leave if they finish the meal (pass serval seconds), the new customer will be added to 3 terminal randomly and if one is full it will added to the empty terminal, once full the system stop.
You can type 'stop' if you want to stop the system. All people will be displayed(include the name, the time which join the restruant and which terminal they are in).
